# 2022-3-31

更换域名 windlinxy.top 并使用阿里 DNS 加速

  • 发现评论跳转有 bug，仔细检查后发现，由于我使用 github.io/blogs ，相当于子目录进行部署博客，在根地址上与生成的评论连接有冲突，查询 js 代码后发现生成的 url 没有带 /blogs/ ，在解析 url 的时候是将 github.io 与 /blogs/ 分离成两个变量，但是，大概是子目录的锅，修改为 github.io 后正常运行。

  • 实现了域名的更换（用另一个域名对源域名做一个映射）

# 2022-6-5

最近收到阿里云的短信，说我域名不合格：

吓了一跳，原来是因为域名与 github 的页面绑定，没有接入阿里的节点（因为都是走 github 的），找了一下解决方法发现，原来域名解析设置一个二级域名绑到自己服务器 ip 上，定时访问一下就行。
